## Todo List Flask App 

## Use

* Flask
* Flask-SQLAlchemy

## Install

```
In Terminal:
pip (or pip3) install -r requirements.txt (this depends on how your system is set up)
```
## Database 

```
In terminal:
$ python 
>>> from app import db
>>> db.create_all()
>>> exit()
```
## Run

```
In Terminal:
python (or python3) app.py (this depends on how your system is set up)
```
